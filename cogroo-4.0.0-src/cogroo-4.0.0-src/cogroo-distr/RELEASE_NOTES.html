Release notes¶

Notas sobre as versões do corretor gramatical CoGrOO. Versão atual versões anteriores

4.0.0 - 30/Maio/2013¶

Esta versão foi reescrita do zero. Todos os modelos linguísticos e dicionários foram refeitos, assim como o duto do corretor gramatical foi aprimorado. Uma nova API foi desenvolvida.
Com as alterações houve uma grande melhoria na qualidade do corretor gramatical, que agora apresenta uma melhor relação entre verdadeiros e falsos positivos.

Novas funcionalidades¶

[*Base*]

  1. O programa foi totalmente redesenhado, desde a arquitetura do software e código até os modelos linguísticos.
  2. Nova API (quebra de compatibilidade com a versão 3.x)
  3. Novo dicionário
  4. Nova linha de comando
[*Regras*]
  1. Aprimoramento das regras sujeito verbo
  2. Regras de regência
  3. Introdução de elementos opcionais

[*Controle de versões*]

  1. Os fontes agora são hospedados no GitHub

Correções¶

Diversas correções e aprimoramentos.

NOTES: Require Java 1.6 or better | Compatible with LibreOffice & Apache OpenOffice (3.0.1 or better)

CHANGE: Completely new development | Dictionary based on Jspell.br | Fewer false positives | Rule improvements

3.2.1 - 12/Outubro/2012¶

Novas funcionalidades¶

[ AddOn ]
  1. No LibreOffice aparece uma nova opção no menu de contexto de um erro gramatical (quando clicamos com o direito no erro com o sublinhado azul). É a opção "Explicações...". Ela vai abrir o browser e mostrar detalhes deste erro no CoGrOO Comunidade. (feature #636)

Correções¶

[ AddOn ]

  1. Corrigimos um bug que impedia reportar erros no Comunidade diretamente do AddOn (bug #634)

[ Regras ]

  1. Melhorias nas regras 17 ... 24 (Concordância adjetivo-substantivo). A regra 17 foi alterada para usar o mecanismo e referencias, e com isso desativamos as regras 18, 19 e 20. O Mesmo foi feito para a regra 20, e desativamos as regras 21, 22, 23 e 24 (bug #633)
  2. Corrigimos um erro que causava exception na regra de pontuação (bug #618)

NOTES: Require Java 1.5 or better | Compatible with LibreOffice & Apache OpenOffice (3.0.1 or better)

CHANGE: Bugs fixing, including one that prevent reporting errors | Rule improvements | Explanations at LibreOffice
