Rules_1.0.xml
	Legacy rules from CoGrOO 1.0 (Perl version).
	Used mainly for the transition from version 1.0 to 2.0.

Rules.xml
	These are the rules used by the grammar checker.

Rules.xsd
	This is the schema that is used to generate and validate the rules used
	by the grammar checker.

Test.xml
	These are the rules used to test the rules applier. The rules are a
	special selection of rules that stresses different aspects of the applier.
